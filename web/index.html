Features

Everything your browser is missing.

📄

Smart Page Reader

Extracts structured content — text, headings, tables, forms — from any page, including complex SPAs.

⚡

Action Engine

Clicks, types, scrolls, navigates, and fills forms on your behalf. Works on any website without plugins.

▶▶

Act Without Asking

Toggle autonomous mode to execute 100+ step tasks end-to-end. Resolve all comments in a doc, pull weekly metrics from 4 tabs, update a codebase from a design screenshot.

🗂

Cross-Tab Synthesis

Reads all your open tabs simultaneously and combines their data into a single report, doc, or answer.

📷

Screenshot Vision

Captures the current tab visually and analyses it. Great for UI feedback, debugging, or matching designs.

🔀

Multi-LLM

Switch between Anthropic Claude, OpenAI GPT-4, and xAI Grok. Your API key, your data, your control.

How it works

Three parts, one seamless experience.

01

Chrome Extension

Side panel UI + service worker that orchestrates the agent loop. Content scripts extract page data and execute DOM actions. Built with React + Vite + TypeScript.

02

Azure Functions Backend

Stateless Python FastAPI backend hosted on Azure Functions Consumption Plan. Handles one LLM turn per request. The loop lives in your browser — no streaming sockets needed.

Setup

Running in under 5 minutes.

1

Clone the repo

git clone https://github.com/venkateshwarreddyr/browser-copilot.git
cd browser-copilot
2

Start the backend

cd backend
python3.12 -m venv .venv && source .venv/bin/activate
pip install -e ".[test,dev]"
# Add your API key to local.settings.json
uvicorn venkat_pilot.app:app --reload --port 7071
3

Build the extension

cd extension
npm install && npm run dev
4

Load in Chrome

Go to chrome://extensions → Enable Developer mode → Load unpacked → select extension/dist/

Open the side panel and enter your LLM API key in Settings.
